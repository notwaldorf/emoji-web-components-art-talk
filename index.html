canvas vs divs

how 2 emojillate
1. render file in a (hidden) <img>
2. find the width/height ratio
3. resize our (hidden) canvas
4. pixellate it! (still hidden)
5. add pixel <divs> somewhere visible
annnnnd it's emojillated!
p.s. it's text!!
so we can make a gist
